# 🐾 Pawsy Platform

## 🎯 Purpose
**Pawsy** is a modern pet adoption and marketplace web application designed to connect helpless animals with loving families. The platform simplifies the pet adoption workflow, enables users to list pets that need a home, and features an integrated marketplace for premium pet care supplies and food, making pet parenting accessible and structured.

---

## 🚀 Live URL
🔗 **Live Application:**  https://pawsy-gamma.vercel.app/

---

## ✨ Features

- **🔐 Dynamic BetterAuth Authentication:** Supports secure Email/Password registration alongside seamless Google Social Login using a dedicated MongoDB adapter.
- **🐕 Interactive Pet Listings:** Authenticated users can list new pets for adoption (`Add Pet`) with detailed descriptions and dynamic status updates.
- **🗂️ Responsive Dashboard Drawer:** Implements a smooth, mobile-friendly side-panel using `HeroUI Drawer` to effortlessly manage dynamic user requests and pet listings.
- **🛒 Premium Supplies Marketplace:** Features an active product showcase on the home page with clean UI grids to display pet care food, toys, and essentials.
- **❤️ Community Engagement Sections:** Includes dedicated extra homepage components for pet adoption success stories and highly informative pet care tips.

## 📦 NPM Packages Used

The core dependencies that power this application include:

- `next` (React Framework for the App Router infrastructure)
- `react` / `react-dom` (Core user interface libraries)
- `@heroui/react` (Component framework used for UI layouts, Avatar, Toast, and Drawer components)
- `better-auth` (Comprehensive client-server authentication toolkit)
- `react-icons` (Icon framework handling design elements like `FaPaw` and dashboard signs)
- `tailwindcss` (Utility-first CSS framework for custom responsive styling)
